# BookNest 📚

A book recommendation app for university students, built with Android + Firebase.

**Individual student project** — Android + Firebase Team Project Sprint (all roles performed solo: Product Owner, Tech Lead, Developer, QA/DevOps).

---

## 📖 Problem Statement

University students struggle to find relevant book recommendations from peers studying similar subjects, so they waste time searching generic book lists that don't match their academic interests.

---

## 🎯 MVP Features

1. **User login** — students sign in with email and password via Firebase Authentication
2. **Browse books** — scrollable list of all book recommendations from Firestore
3. **View book details** — tap any book to see title, author, description, genre, and recommender
4. **Add a book recommendation** — form to share a book with other students
5. **Save to "Want to Read"** — bookmark books for later

## 🗄️ Data Model — Firestore Collections

### `users`
| Field | Type | Description |
|-------|------|-------------|
| userId | string | Auto-generated by Firebase Auth |
| email | string | Login email |
| displayName | string | Public name shown in the app |
| university | string | e.g. Warsaw University of Technology |
| createdAt | timestamp | Account creation date |

### `books`
| Field | Type | Description |
|-------|------|-------------|
| bookId | string | Auto-generated document ID |
| title | string | Book title |
| author | string | Book author |
| description | string | Short summary (~500 chars max) |
| genre | string | e.g. Fiction, Science, Biography |
| recommendedBy | string | Reference to userId |

### `savedBooks`
| Field | Type | Description |
|-------|------|-------------|
| savedId | string | Auto-generated document ID |
| userId | string | Reference to users collection |
| bookId | string | Reference to books collection |
| savedAt | timestamp | When the book was saved |
| status | string | "want-to-read" or "finished" |
